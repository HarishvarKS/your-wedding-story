

## Wedding Photography Homepage

A stunning, cinematic wedding photography homepage inspired by the reference image, with a black & white elegant color scheme and full interactive animations.

### Sections (top to bottom)

1. **Hero Section — Auto-playing Image Slideshow**
   - Full-screen slideshow with 5-6 beautiful wedding placeholder images that auto-rotate every 4 seconds with smooth crossfade transitions
   - Transparent navigation bar overlaid on top: Logo (left), nav links Home / Portfolio / About / Contact (right)
   - Overlay text with studio tagline, subtle fade-in animation

2. **Values Section — Art / Emotions / Story**
   - Three columns with elegant hand-drawn style illustrations (using line-art SVG icons)
   - Each column: icon, title (ART, EMOTIONS, STORY), descriptive paragraph
   - Scroll-triggered fade-in animations, staggered per column

3. **Our Offerings Section**
   - Dark/black background with white text for contrast
   - Section title "Our Offerings" with category tags (Weddings, Engagements, Destinations, Families, Commercials)
   - Descriptive paragraph about services
   - Scroll-triggered slide-up animation

4. **Approach & Style Section**
   - Two-column layout: large image on left, text content on right
   - Multiple paragraphs about the photography approach and style
   - Parallax effect on the image as user scrolls

5. **Services Cards Section**
   - Three image cards in a row: Wedding Films, Wedding Documentaries, Maternity & Milestones
   - Hover effects: image zoom, overlay text reveal with scale animation
   - Each card has a label overlay at the bottom

6. **Founder Section**
   - Two-column layout: portrait image on left, bio text on right
   - Name, title "Founder + Storyteller", short bio paragraph
   - "Read More" link with underline animation on hover

7. **Footer**
   - Dark background with centered text about social links (YouTube, Instagram) and contact form reference
   - Copyright line at the bottom

### Design & Interactions
- **Color palette**: Pure black & white with subtle gray gradients for depth
- **Typography**: Elegant serif font for headings, clean sans-serif for body text
- **Scroll animations**: Every section fades/slides in on scroll using Intersection Observer
- **Hover effects**: Images scale on hover, buttons/links have underline animations
- **Parallax**: Background images in key sections move at different scroll speeds
- **Hero slideshow**: Smooth crossfade with progress indicator dots, auto-advances every 4s, pausable on hover

### Images
- Using high-quality placeholder images from Unsplash (wedding photography) for all sections
- Line-art SVG illustrations for the Art/Emotions/Story icons
